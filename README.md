# Busca endereço por CEP
Faça suas consultas individuais de CEP, obtendo o retorno do endereço completo. Com o APP Busca endereço por CEP é possível descobrir de qual cidade e estado é o CEP e até mesmo qual o nome do logradouro.

## Link do APP na Google Play
https://play.google.com/store/apps/details?id=br.com.andersonmatte.buscacep

## Link da API ViaCep
