# Busca endereço por CEP

![alt text](/Imagens/Capa.png)

Faça suas consultas individuais de CEP, obtendo o retorno do endereço completo.
Com o APP Busca endereço por CEP é possível descobrir de qual cidade e estado é o CEP e até mesmo qual o nome do logradouro. Você poderá realizar a sua busca por CEP ou por endereço.
